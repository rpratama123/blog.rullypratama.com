For me, *dark mode* is more comfortable and looks better than *light mode*. But unfortunately, many applications I use have their own separate *dark* versions. That's until I found the [Catppuccin](https://github.com/catppuccin/) themes which are available for many applications. As a result, the application interface becomes coherent and looks "unified."

This post will be updated periodically.

## Firefox
{{< img src="Firefox.webp" width="800" alt="Catppuccin theme for Firefox" caption="Catppuccin theme for Firefox." >}}

* [Cascade userChrome + Catppuccin theme](https://github.com/andreasgrafen/cascade)
* The userChrome I use can be downloaded [here](/assets/downloads/dark-themes/Rully's-chrome-folder.zip), with the installation method [like this](https://github.com/andreasgrafen/cascade#how-to-install-cascade). But please forgive me if my userChrome is messy 🤣
