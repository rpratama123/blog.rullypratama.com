There are many new design aspects in Windows 11 compared to previous Windows versions. One of them is the emojis. Unfortunately, these new emojis were not included in the retail version of Windows 11 released on October 10, 2021. Fortunately, the new emojis from the *Insider Preview* version can be installed on the retail version of Windows 11, and even on Windows 10.

In Windows, emojis are stored in the `Segoe UI Emoji` font file with a `.ttf` extension. As such, font files from one version of Windows can be installed on another version of Windows. Below, I've provided the emoji font from Windows 11 Insider Preview 22478.1000.

1. Download the emoji font in a `.zip` file through [the following link](/assets/downloads/win11-emoji/segoe-fonts.zip).
2. Extract the downloaded `.zip` file.
3. Select all `.ttf` files, right-click, and select **Install for all users**.
4. If a confirmation dialog box appears to replace already installed fonts, click **Yes**.
5. Restart your computer for the changes to take effect.
