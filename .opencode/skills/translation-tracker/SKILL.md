# Translation Tracker

This skill manages the translation status of all blog posts.

## When to Use

Load this skill when:
- Translating a post (check status before, update after)
- Reviewing which posts need translation
- Reporting translation coverage to the user

## Reference File

The full translation tracker is in `.opencode/refs/translation-tracker.md`.
Read and update that file as needed.
